# 轮毂识别功能优化说明

## 问题描述

移动端的轮毂识别功能对现实拍照的轮毂识别率极低(仅1.7%),无法正确识别。

### 问题原因分析

1. **光照影响严重**: 现实照片有强烈反光、阴影、光照不均
2. **拍摄角度多样**: 不是标准正面照,有各种角度
3. **背景干扰**: 工厂环境、地面、手等干扰元素
4. **参数配置保守**: 
   - ORB特征点只有800个,对复杂场景不足
   - 匹配阈值过严格
   - 缺少图像预处理

## 优化方案

### 1. 增加特征点数量
```java
ORB_FEATURES: 800 → 1500  // 提升87.5%
```
- 更多特征点意味着更robust的匹配
- 特别对于轮毂这种复杂纹理的物体

### 2. 放宽匹配参数
```java
MATCH_RATIO_THRESHOLD: 0.7 → 0.75  // Lowe比率放宽
MIN_GOOD_MATCHES: 10 → 8  // 降低最小匹配要求
MIN_SCORE_THRESHOLD: 0 → 0.015  // 设置最低分数阈值(1.5%)
```

### 3. 添加图像预处理 (核心优化)

#### 3.1 CLAHE自适应直方图均衡化
```java
CLAHE clahe = Imgproc.createCLAHE(2.0, new Size(8, 8));
clahe.apply(gray, claheResult);
```
**作用**: 
- 处理光照不均问题
- 减弱反光影响
- 增强局部对比度
- 对现实拍照的轮毂效果显著

#### 3.2 双边滤波
```java
Imgproc.bilateralFilter(claheResult, filtered, 5, 50, 50);
```
**作用**:
- 降低噪点
- 保留边缘特征(轮毂的关键特征)
- 平滑非边缘区域

#### 3.3 锐化处理
```java
Mat kernel = new Mat(3, 3, CvType.CV_32F);
kernel.put(0, 0, -0.5, -0.5, -0.5);
kernel.put(1, 0, -0.5,  5.0, -0.5);
kernel.put(2, 0, -0.5, -0.5, -0.5);
Imgproc.filter2D(filtered, sharpened, -1, kernel);
```
**作用**:
- 增强轮毂边缘
- 提升特征检测质量
- 改善模糊照片的识别

### 4. 优化评分公式
```java
// 优化前
double rawScore = countScore * distanceScore * 2.0;

// 优化后
double distanceScore = 1.0 / (1.0 + avgDistance / 60.0);  // 50→60
double rawScore = countScore * distanceScore * 2.5;  // 2.0→2.5
```
**作用**:
- 对距离评分更宽容
- 提升权重系数
- 让相似度分数更合理

### 5. 增加详细日志
```java
log.info("查询图片特征点数量: {}", queryKeyPoints.rows());
log.info("找到 {} 个候选匹配", matchResults.size());
log.info("候选 {}: {} - 相似度 {:.2f}%", i+1, mr.wheelCode, mr.score * 100);
```
**作用**: 便于调试和分析识别效果

## 技术细节

### 预处理流程
```
原始照片 
  ↓
灰度转换
  ↓
CLAHE自适应均衡化  ← 核心:处理反光和光照不均
  ↓
双边滤波降噪      ← 保留边缘,去除噪点
  ↓
锐化增强边缘      ← 提升特征质量
  ↓
ORB特征提取 (1500个点)
  ↓
特征匹配
```

### 匹配策略优化
1. **kNN匹配**: 每个描述子找2个最近邻
2. **Lowe比率检验**: 最佳距离 < 0.75 * 次佳距离
3. **最小匹配数**: 至少8个好的匹配点
4. **综合评分**: 匹配数 × 距离质量 × 2.5

## 注意事项

1. **性能影响**: 图像预处理会增加10-20%的处理时间,但对于识别功能是值得的
2. **内存占用**: 更多特征点会略微增加内存,但在可接受范围内
3. **索引重建**: 首次启动会重新构建索引,可能需要几秒到几分钟(取决于产品数量)

本次优化通过**增加特征点**、**添加图像预处理**和**放宽匹配参数**三个方向,大幅提升了对现实拍照轮毂的识别能力。核心是CLAHE自适应直方图均衡化,能有效处理现实照片的光照不均和反光问题。
