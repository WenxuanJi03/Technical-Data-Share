# 快速测试指南

## 测试步骤

### 1. 后端编译测试

```bash
cd ruoyi-admin
mvn clean compile
```

如果编译成功,说明后端代码没有语法错误。

### 2. 启动后端服务

```bash
# 在 ruoyi-admin 目录下
mvn spring-boot:run
```

或者在IDE(如IntelliJ IDEA)中直接运行 `RuoYiApplication` 主类。

### 3. 测试后端接口

启动后,可以使用浏览器或Postman测试接口:

```
GET http://localhost:8080/tech/dashboard/allStats
```

预期返回JSON格式的统计数据:
```json
{
  "msg": "操作成功",
  "code": 200,
  "data": {
    "trialTrack": {
      "total": 数量,
      "moldTypeStats": [...],
      "surfaceStatusStats": [...],
      "machineTypeStats": [...]
    },
    "product": {
      "total": 数量,
      "productStatusStats": [...],
      "customerStats": [...],
      "sizeSpecStats": [...]
    }
  }
}
```

### 4. 启动前端开发服务器

```bash
cd ruoyi-ui
npm run dev
```

### 5. 访问首页

在浏览器中打开: `http://localhost:80` (或前端配置的端口)

登录后应该能看到新的数据看板页面。

## 可能遇到的问题

### 问题1: 图表不显示

**原因**: 可能是数据库中没有数据

**解决方法**:
1. 先导入一些OE试制跟踪表数据
2. 导入一些产品清单数据
3. 刷新首页

### 问题2: 后端编译错误

**原因**: 可能是依赖问题

**解决方法**:
```bash
mvn clean install
```

### 问题3: 接口返回401未授权

**原因**: 接口需要登录后访问

**解决方法**:
1. 先登录系统
2. 或者在 Spring Security 配置中添加匿名访问权限(不推荐)

### 问题4: 前端报错找不到dashboard.js

**原因**: 新创建的API文件未被识别

**解决方法**:
1. 重启前端开发服务器
2. 清除浏览器缓存

## 测试数据准备

如果数据库中没有测试数据,可以通过以下方式添加:

### 方式1: 通过界面导入

1. 访问 "OE试制跟踪" 菜单
2. 点击 "导入" 按钮
3. 上传Excel文件

4. 访问 "产品清单" 菜单
5. 点击 "导入" 按钮
6. 上传Excel文件

### 方式2: 通过SQL直接插入

```sql
-- 插入测试数据到 tech_trial_track 表
INSERT INTO tech_trial_track (mold_code, mold_type, surface_status, machine_type) VALUES
('M001', '首模', '精车', '小批量'),
('M002', '改模', '全涂', '量产'),
('M003', '翻模', '全涂装', '小批量');

-- 插入测试数据到 tech_product 表
INSERT INTO tech_product (wheel_code, customer, product_status, size_spec) VALUES
('W001', '丰田', '量产', '1775'),
('W002', '本田', '试制/小批量', '1880'),
('W003', '日产', '退市/停产', '1990');
```

## 预期效果

成功部署后,首页应该显示:

1. **顶部标题区**
   - 紫色渐变背景
   - "技术科数据分析看板" 标题

2. **OE试制跟踪统计区** (3个图表)
   - 模具类型分布 (环形饼图)
   - 表面状态分布 (环形饼图)
   - 上机类型数量 (柱状图)

3. **产品清单统计区** (3个图表)
   - 产品状态分布 (环形饼图)
   - 客户分布 (环形饼图)
   - 轮毂规格数量 (柱状图)

每个图表卡片都应该:
- 显示标题和总数
- 有悬停动效
- 图表可交互(点击图例、鼠标悬停提示)

## 性能检查

1. **首次加载速度**: 应在2秒内完成
2. **图表渲染**: 流畅无卡顿
3. **窗口缩放**: 图表自动适配
4. **内存占用**: 打开F12查看,应无明显内存泄漏

## 浏览器兼容性

推荐使用以下浏览器测试:
- Chrome 90+
- Edge 90+
- Firefox 88+
- Safari 14+

不支持 IE 浏览器。
